## Common Features

All SDKs share these common features:

### Configuration

Initialize the client with your credentials:

<Tabs>
  <Tab title="Node.js">
    ```javascript
    const { DoorPassesClient } = require('@doorpasses/node-sdk');

    const client = new DoorPassesClient({
      accountId: 'your-account-id',
      sharedSecret: 'your-shared-secret',
      baseUrl: 'https://api.doorpasses.com' // Optional
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from doorpasses import DoorPassesClient

    client = DoorPassesClient(
        account_id='your-account-id',
        shared_secret='your-shared-secret',
        base_url='https://api.doorpasses.com'  # Optional
    )
    ```
  </Tab>
  <Tab title="PHP">
    ```php
    use DoorPasses\DoorPassesClient;

    $client = new DoorPassesClient([
        'accountId' => 'your-account-id',
        'sharedSecret' => 'your-shared-secret',
        'baseUrl' => 'https://api.doorpasses.com'  // Optional
    ]);
    ```
  </Tab>
  <Tab title="Go">
    ```go
    import "github.com/doorpasses/go-sdk"

    client := doorpasses.NewClient(
        "your-account-id",
        "your-shared-secret",
        doorpasses.WithBaseURL("https://api.doorpasses.com"),
    )
    ```
  </Tab>
</Tabs>

### Access Pass Management

Create, update, and manage access passes:

```javascript
// Issue a new access pass
const pass = await client.accessPasses.create({
  card_template_id: '0xt3mp14t3',
  full_name: 'Ahmed Al-Mansouri',
  email: 'ahmed@company.com',
  // ... other fields
});

// List access passes
const passes = await client.accessPasses.list({
  template_id: '0xt3mp14t3',
  state: 'ACTIVE'
});

// Update an access pass
await client.accessPasses.update('0xp4551d', {
  title: 'Lead Engineer'
});

// Suspend an access pass
await client.accessPasses.suspend('0xp4551d');

// Resume an access pass
await client.accessPasses.resume('0xp4551d');

// Delete an access pass
await client.accessPasses.delete('0xp4551d');
```

### Error Handling

All SDKs provide structured error handling:

<Tabs>
  <Tab title="Node.js">
    ```javascript
    try {
      const pass = await client.accessPasses.create(data);
    } catch (error) {
      if (error.code === 'VALIDATION_ERROR') {
        console.error('Validation failed:', error.details);
      } else if (error.code === 'UNAUTHORIZED') {
        console.error('Authentication failed');
      } else {
        console.error('Error:', error.message);
      }
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    try:
        pass = client.access_passes.create(data)
    except ValidationError as e:
        print(f"Validation failed: {e.details}")
    except UnauthorizedError:
        print("Authentication failed")
    except DoorPassesError as e:
        print(f"Error: {e.message}")
    ```
  </Tab>
</Tabs>

## Environment Variables

For security, we recommend storing credentials in environment variables:

```bash
export DOORPASSES_ACCOUNT_ID="your-account-id"
export DOORPASSES_SHARED_SECRET="your-shared-secret"
```

Then use them in your code:

```javascript
const client = new DoorPassesClient({
  accountId: process.env.DOORPASSES_ACCOUNT_ID,
  sharedSecret: process.env.DOORPASSES_SHARED_SECRET
});
```
