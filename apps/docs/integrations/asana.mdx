# Asana Integration

Connect your Epic Stack workspace to Asana to manage projects and tasks seamlessly.

## Overview

The Asana integration allows you to:

- Link notes to Asana tasks
- Create tasks from notes
- Sync task status and updates
- Manage projects and teams
- Track work across platforms

## Setup Instructions

### 1. Create an Asana OAuth Application

1. Go to [Asana Developer Console](https://app.asana.com/0/developer-console)
2. Click **Create new app**
3. Fill in the application details:
   - **App name**: Epic Stack
   - **Redirect URL**: `https://your-domain.com/api/integrations/oauth/callback`
4. Accept the terms and click **Create app**
5. Copy your **Client ID** and **Client Secret**

### 2. Set Environment Variables

Add these to your `.env` file:

```bash
ASANA_CLIENT_ID=your_asana_client_id
ASANA_CLIENT_SECRET=your_asana_client_secret
INTEGRATION_ENCRYPTION_KEY=64_character_hex_string
```

### 3. Connect Your Asana Account

1. Go to your organization settings
2. Navigate to the **Integrations** tab
3. Find **Asana** and click **Connect**
4. You'll be redirected to Asana's authorization page
5. Log in with your Asana account
6. Click **Authorize** to grant permissions

### 4. Select Projects

After connecting, select which Asana projects you want to work with:

- Choose default projects for task creation
- Configure default assignees and sections
- Set up project-specific workflows

## Using the Integration

### Linking Notes to Tasks

1. Open a note in Epic Stack
2. Click the **Link to Asana** option
3. Search for or select an Asana task
4. The note will be linked to the task

### Creating Tasks from Notes

1. Create a note in Epic Stack
2. Click **Create Asana Task**
3. Select the project
4. Configure task details (name, description, assignee, due date, priority, etc.)
5. Click **Create**
6. The task will be created in Asana and linked to your note

### Syncing Task Status

Connected tasks automatically sync:

- Status changes in Asana appear in Epic Stack
- Comments and updates are reflected
- Assignee changes are synchronized

## Advanced Features

### Custom Fields

Map Epic Stack fields to Asana custom fields:

- Configure field mappings in integration settings
- Sync custom field values automatically
- Support for various field types

### Workflow Automation

Set up automated workflows:

- Auto-create tasks when notes reach certain status
- Update task status based on note changes
- Trigger notifications on task updates

## Troubleshooting

### Connection Issues

**Problem**: Authorization fails or shows permission errors

**Solution**:
- Ensure you have admin access to the Asana workspace
- Check that your Asana account has API access enabled
- Verify your Asana workspace is active
- Try disconnecting and reconnecting

### Tasks Not Appearing

**Problem**: Expected Asana tasks don't show up in Epic Stack

**Solution**:
- Verify you have permission to view the tasks in Asana
- Check that the project is selected in integration settings
- Refresh the integration connection
- Ensure the task type is supported

### Sync Failures

**Problem**: Changes aren't syncing between platforms

**Solution**:
- Check your internet connection
- Verify the integration is still connected
- Review Asana API rate limits
- Reconnect the integration

## Security & Privacy

- All communication with Asana is encrypted
- Your Asana tokens are securely stored and encrypted
- Epic Stack only accesses the permissions you explicitly grant
- You can revoke access at any time from Asana's app management

## Disconnecting

To disconnect Asana:

1. Go to organization settings
2. Navigate to **Integrations**
3. Find **Asana** and click **Disconnect**
4. Confirm the disconnection

You can also revoke access from your Asana account settings.
